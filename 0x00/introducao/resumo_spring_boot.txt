O que é Spring Initializr?

Spring Boot Framework é um framework para desenvolvimento de microsserviços.
Alguns benefícios:
Inicialização rápida, configuração automática e servidor embutido.
As dependências podem ser adicionadas atraés do Spring Initializr, maven ou gradle.
A melhor IDE é o Spring Tools Suites.
Spring Initializr é uma ferramenta que permite criar projetos Spring Boot com suas dependências.
